### 环境要求

* 站点运行环境要求 PHP 7.2 及以上，推荐 7.4 或 8.1
* MySQL 5.6 及以上
* 服务器环境推荐：Linux + Nginx

## 本地开发与验证说明

- 当前这台开发机可以使用 `php` CLI。
- 本地可执行 `php -l`、`php -v` 等命令做开发期预检查。
- 由于本地与服务器 PHP 版本可能不同，涉及 PHP 文件修改后的最终语法校验，仍需在服务器上进入站点目录后逐个执行 `php -l`。
