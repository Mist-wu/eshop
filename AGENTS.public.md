# AGENTS.public.md

本文件是公开仓库可保留的协作说明，不包含任何私有运维信息。

## 开源边界

- 严禁把真实线上域名、服务器 IP、SSH 用户名、口令、后台账号、API Token、支付私钥提交到 Git
- `.env` 只用于本地敏感配置，仓库中只保留 `.env.example`
- 本地私有运维信息保留在未跟踪的 `AGENTS.md`

## 项目事实

- 本地可使用 `php` CLI 做预检查，修改 PHP 后优先执行 `php -l`
- 支付相关 API 优先参考易付通官方文档：`https://www.yifut.com/doc/index.html`
- 运行时目录与本地测试产物不应入库，例如缓存、上传、日志、临时文件、`test-results`

## 变更原则

- 优先做小范围、可审阅的改动
- 不要把调试日志、抓包结果、浏览器自动化产物、数据库导出、私有截图提交到仓库
- 对外说明写在公开文档中；仅限本地使用的信息留在未跟踪文件中

## 验证建议

- 对本次修改过的 PHP 文件逐个执行 `php -l`
- 涉及支付、后台或发布流程的改动，优先补充最小可复现的验证步骤
