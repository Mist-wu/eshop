# AGENTS.public.md

公开仓库协作说明，不包含任何私有运维信息。本机私有信息请放在未跟踪的 `AGENTS.md`。

## 开源边界

- 不要把真实域名、服务器 IP、SSH 凭据、后台账号、API Token、支付私钥提交到 Git
- `.env` 只用于本地敏感配置，仓库内只保留 `.env.example`
- 缓存、上传、日志、临时文件、`test-results` 等运行时产物不入库
- 不要把调试日志、抓包结果、浏览器自动化产物、数据库导出、私有截图提交到仓库

## 项目事实

- 本地可使用 `php` CLI；修改 PHP 文件后优先执行 `php -l`
- 线上目标 PHP 版本为 8.1，新写法需兼容
- 安装入口和上传接口已移除，不要再恢复
- 可见品牌名统一为大写 `ESHOP`，包名 / 路径 / 标识符仍用小写 `eshop`
- 支付相关 API 优先参考易付通官方文档：<https://www.yifut.com/doc/index.html>

## 变更原则

- 优先做小范围、可审阅的改动
- 优先编辑现有文件，不随手新建文档
- 涉及支付、后台或发布流程的改动，补充最小可复现的验证步骤
- 后台 AJAX / 表单写操作必须调用 `LoginAuth::checkToken()`；对应请求携带 `token` 字段，值来自 `LoginAuth::genToken()`

## 验证建议

- 对本次修改过的 PHP 文件逐个执行 `php -l`
- 涉及前台模板的改动，至少手动打开 `/` 和商品详情页确认无 500 / 白屏
